Viewing and filtering all Shipping Rules
----------------------------------------

To view all Shipping Rules, navigate to **System > Shipping Rules** in the main menu.

**Next steps**

You can perform the following actions with every item in the Shipping Rules list:

 * View

 * Edit

 * Enable

 * Disable

 * Delete


You can perform the following actions at the Shipping Rules page:

 * Create Shipping Rule
