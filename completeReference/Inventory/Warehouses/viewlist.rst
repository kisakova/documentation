Viewing and filtering all Warehouses
------------------------------------

To view all Warehouses, navigate to **Inventory > Warehouses** in the main menu.

Preview:

.. include:: /completeReference/overview/Inventory/WarehousesOverview.rst
  :start-after: begin


.. image:: /completeReference/img/Inventory/Warehouses/Warehouses.png
   :class: with-border

The following information about the Warehouses is available in the Warehouses list:

+------------+-------------+
| Name       | Description |
+============+=============+
| NAME       |             |
+------------+-------------+
| CREATED AT |             |
+------------+-------------+
| UPDATED AT |             |
+------------+-------------+

**Next steps**

You can perform the following actions with every item in the Warehouses list:

 * Edit

 * Delete

 * View


You can perform the following actions at the Warehouses page:

 * Create Warehouse
